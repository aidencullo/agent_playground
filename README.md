# Claude Code Playground

Access point for Claude Code web to build new projects.

This repository serves as a starting point for developers using Claude Code to rapidly prototype and build new applications. It provides a clean, minimal foundation for project development with integrated support for Claude Code workflows.

## Quick Start

1. Clone this repository
2. Run `bun install` to install dependencies
3. Start building with Claude Code

## Features

- Minimal, clean project structure
- Ready for Claude Code integration
- Built for rapid prototyping
- Bun runtime support

## Usage

This repo is designed to be used as a template or starting point for new projects initiated through Claude Code web interface.

## Documentation

For more information on using Claude Code, visit the [Claude Code documentation](https://github.com/anthropics/claude-code).
